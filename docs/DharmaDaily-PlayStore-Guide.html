4. Play Store Submission Checklist

Pre-Submission Requirements
  • [ ] Google Play Developer account created & verified ($25)
  • [ ] App icon: 512x512 PNG, no transparency, no rounded corners
  • [ ] Feature graphic: 1024x500 PNG
  • [ ] Phone screenshots: Min 2, recommended 8 (1080x1920 or 1440x2560)
  • [ ] 7-inch tablet screenshots: Min 2 recommended
  • [ ] Short description: Max 80 characters
  • [ ] Full description: Max 4000 chars (Telugu + English keywords)
  • [ ] Privacy Policy URL (hosted, publicly accessible)
  • [ ] App signed with upload key (EAS handles this)
  • [ ] AAB (Android App Bundle) built successfully
Play Console Setup
  • [ ] App category: Lifestyle or Books & Reference
  • [ ] Content rating: Complete IARC questionnaire (expect "Everyone")
  • [ ] Target audience: General (NOT children-directed to avoid COPPA)
  • [ ] Data Safety form completed
  • [ ] Pricing: Free (with in-app purchases later)
  • [ ] Countries: India (primary), USA, UK, UAE, Singapore, Australia

Mandatory 20-Tester Closed Testing (14 days)

Google requires 20 unique testers who have opted in to your closed testing track for at least 14 consecutive days before you can publish to production.

  • Recruit from: Family, friends, Telugu community WhatsApp groups
  • Create a Google Group or email list for testers
  • Share the opt-in link from Play Console
  • Ask testers to install and use the app daily
  • Collect feedback via Google Forms

7. Legal & Tax Compliance

  • GST Registration — Required if annual revenue exceeds Rs.20 lakhs
  • TDS — Google deducts TDS on Indian developer earnings
  • PAN — Required for Google Play Developer verification
  • Bank Account — Add to Payments Center for payouts (minimum Rs.100)
  • Image Licensing — All Wikimedia images must have attribution; verify each is CC BY-SA or Public Domain
  • No Copyrighted Content — Ensure stories and descriptions are original or public domain
  • AdMob Policy — Never click own ads, never encourage ad clicks, never place ads that mimic content

8. Key Commands Reference

# Install dependencies
npm install

# Install EAS CLI
npm install -g eas-cli

# Login to Expo
eas login

# Configure EAS build
eas build:configure

# Build Android App Bundle (production)
eas build --platform android --profile production

# Build for testing (APK)
eas build --platform android --profile preview

# Submit to Play Store
eas submit --platform android

# Start local dev server
npx expo start
